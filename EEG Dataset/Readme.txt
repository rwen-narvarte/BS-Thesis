FILE OVERVIEW
----------------------
Text files (.txt). 
Files: For each set (A-E) there is a ZIP-file containing 100 TXT-files. 
Each TXT-file consists of 4096 samples of one EEG time series in ASCII code. SET A in file Z.zip containing Z000.txt - Z100.txt, SET B in file O.zip containing O000.txt - O100.txt, SET C in file N.zip containing N000.txt - N100.txt, SET D in file F.zip containing F000.txt - F100.txt, SET E in file S.zip containing S000.txt - S100.txt
